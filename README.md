## Перед началом работы

Установите [nvm](https://github.com/nvm-sh/nvm#installation-and-update) (Node.js version manager)

```bash
$ nvm install 18
$ npm install
```

## Работа над клентом

```bash
$ API_URL=http://localhost:8000 npm start
```

## Сборка проекта

```bash
$ API_URL=https://quiz-on.ru/api npm run build
```
